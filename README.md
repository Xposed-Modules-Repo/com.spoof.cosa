# CosaSpoof

LSPosed/Xposed Module Repository 发布仓库。

源码、问题反馈和完整说明请看：<https://github.com/wowohut/cosa-cloudcontrol-spoofer>

如果这个模块对你有帮助，欢迎点个 Star ⭐，这是我持续更新的动力。

## 功能

- 伪装 `com.oplus.cosa` 读取的 `ro.boot.prjname` 机型代号，以获取目标机型的游戏云控配置。
- 支持在模块设置中配置目标机型代号。
- 在相同 SoC 平台或相近机型上，可能获得更合适的游戏调度策略与体验。
