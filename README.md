# CosaSpoof

LSPosed/Xposed Module Repository 发布仓库。

源码、问题反馈和完整说明请看：<https://github.com/wowohut/cosa-cloudcontrol-spoofer>

## 功能

- 拦截 ColorOS / realmeUI 应用增强服务 `com.oplus.cosa` 读取 `ro.boot.prjname`
- 将设备主板机型代号伪装为用户配置的目标机型代号
- 用于获取目标机型的游戏云控 / 性能调度配置
- 提供 Compose 设置界面，支持配置伪装代号
